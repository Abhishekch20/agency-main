# Frontend

This website is frontend-only and does not require the `backend` folder to run or deploy.

## Scripts

### `npm start`

Runs the site locally at [http://localhost:3000](http://localhost:3000).

### `npm run build`

Builds the production bundle into `build/`.

### `npm test`

Runs the test suite.

## Contact Form

The contact form works without a backend. Submitting it opens the visitor's default email app with a prefilled draft addressed to `support@example.com`.
